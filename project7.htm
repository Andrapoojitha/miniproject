    • HTML
    HTML stands for Hypertext Markup Language. It is the standard language used to create and structure web pages and their content for display in a web browser. HTML uses tags to organize text, images, and other media, defining the layout and meaning of web content for browsers to interpret.
    CSS
    CSS stands for Cascading Style Sheets. It is a stylesheet language used to describe the presentation and formatting of a document written in HTML or XML. CSS allows web developers to control the layout, colors, fonts, and overall visual appearance of web pages, enabling them to create visually appealing and consistent designs across multiple pages.
    JS
    JavaScript (JS) is a versatile programming language primarily used for creating interactive and dynamic content on websites. It enables developers to implement features such as animations, form validation, and real-time updates, enhancing user experience. JavaScript can be executed on both the client-side (in web browsers) and server-side (using environments like Node.js), making it a fundamental technology for modern web development.
